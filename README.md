# Thermosight
HVAC Energy Saving Consultancy

Thermosight installs sensory devices in your building and collects HVAC performance data to be analyzed. Once collected, a CSV file is uploaded and inisights are provided so that users can reduce wasted energy costs.

Site has diagnostic appointment booking form, login system, and dashboard of all insights (charts, graphs, etc.) and previous uploads.

How to run:
1. Install dependecnies: pip install -r requirements.txt
2. Run application in terminal: python app.py
